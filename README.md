# Employee-Project-Analytics-System

# Project Objective

The Employee & Project Analytics System project aims to analyze employee, department, and project data to provide insights into workforce productivity, project allocation, department budgets, and overall organizational efficiency. Key objectives include tracking hours worked, salaries, project budgets, and departmental performance.

Dataset Used:
- <a href="https://github.com/eshita100/Employee-Project-Analytics-System/blob/main/Employ_management.sql">Dataset</a>

Employees Table: employee ID, name, department ID, salary, hire date

Departments Table: department ID, department name

Projects Table: project ID, project name, department ID, budget

Employee_Projects Table: employee ID, project ID, hours worked


## Questions / KPIs

The system answers several key questions about employees, projects, and departments:

Total hours worked by each employee across projects

Average hours per project and per employee

Highest-paid employee in each department

Employees who joined before 2020

Departments with no projects assigned

Total number of projects per department

Departments with average project budget exceeding a certain threshold

Employees who worked more than a specific number of hours

Departments with projects under a certain budget

## Process

Data Cleaning & Verification: Checked for missing or inconsistent values in employee, department, project, and assignment data.

Analytical Queries: Designed SQL queries to answer the KPIs and identify trends in employee productivity and project allocation.

Reporting: Prepared detailed documentation summarizing departmental performance, total project hours, and individual employee contributions.

## Project Insights

Some employees contribute significantly more hours to multiple projects, highlighting key resources.

Certain departments have no projects assigned, indicating potential planning gaps.

Highest-paid employees are concentrated in specific departments.

Average project budgets vary widely across departments, helping identify over- or under-budgeted areas.

Departments with projects below a certain budget can be targeted for review.
